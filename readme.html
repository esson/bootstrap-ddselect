bootstrap-ddselect.js
Transform HTML Select elements to Bootstrap Split Button Dropdowns

About

This jQuery plugin extends the Twitter Bootstrap front-end framework to transform <select> elements to split button dropdowns (or dropups).

Version
1.0
Author
Martin Eriksson

Features

  • Supports the bootstrap default button styles.
  • Supports option groups.

Usage

Via JavaScript

Call the dropdowns via JavaScript:

$('select').ddselect();

Options

Options can be passed via JavaScript.

Name type default description
buttonClass string Use any of the bootstrap button style class names to style your dropdown. Class names are .btn-primary, .btn-info, .btn-success, .btn-warning, .btn-danger and .btn-inverse,
sizeClass string Use any of the bootstrap button size class names to size your dropdown. Class names are .btn-mini, .btn-small and .btn-large.
dropup bool false Set to true to transform the dropdown into a dropup.

Methods

$().ddselect('refresh')

Refreshes the dropdown completely. Use this if you have changed anything but the selected value of the original <select> element to update the dropdown.

$().ddselect('show')

Show the dropdown.

$().ddselect('hide')

Hide the dropdown.
